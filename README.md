# Plot Viewer

Simple application for displaying PNG plots generated with [matplotlib pyplot](https://matplotlib.org/3.2.1/api/_as_gen/matplotlib.pyplot.html) from my engineering thesis.

Contains sample plots.

## Installation
>`npm install`

## Run:
>`npm start`
